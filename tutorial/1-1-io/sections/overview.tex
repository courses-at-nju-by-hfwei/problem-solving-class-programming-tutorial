\begin{frame}{C 还是 C++?}
  \begin{columns}
    \column{0.45\textwidth}
      C 语言是基础：
      \begin{itemize}
	\item IO
	\item Control Flow
	\item Function
	\item Array (String) \& Pointers
	\item Struct
      \end{itemize}
    \column{0.55\textwidth}
      C++ 语言的特性：
      \begin{itemize}
	\item OO (Object-oriented)
	\item Templates
	\item STL (Standard Template Library)
	\item FP (Functional Programming)
	\item Concurrency
      \end{itemize}
  \end{columns}

  \vspace{1.00cm}
  \centerline{先学习 C 语言; 是否学习 C++ 视情况而定。}
\end{frame}
